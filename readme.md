# wechart mini app -- gulp + typescript construction 
# 微信小程序 typescript + gulp 构筑
## prerequisite

> wechart mini app IDE

> node.v >= 4.x | 6.x

> typescript >= 2.0
```
# npm install -g gulp-cli typescript tslint
```

## init 
``` bash
# npm install
```

## dev
``` bash
# gulp
or 
# npm start
```

Then run your IDE and open ```./dist``` to debug page.
